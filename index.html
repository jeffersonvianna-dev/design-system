🔘 Botões

Três variantes principais: primary (ação principal da tela), ghost (ação secundária/cancelar) e danger (excluir). Ícone para ações de tabela. Modifier .c-btn-sm para versões compactas.

Variantes
Tamanho pequeno (.c-btn-sm)
Ícone — ações de tabela
Use sempre em pares (editar + excluir) na coluna "Ações" da tabela. Não adicione texto.
/* Primary — ação principal */ <button class="c-btn c-btn-primary">+ Nova Aula</button> /* Ghost — cancelar / ação secundária */ <button class="c-btn c-btn-ghost">Cancelar</button> /* Danger — excluir */ <button class="c-btn c-btn-danger">Excluir</button> /* Ícone — coluna Ações da tabela */ <button class="c-btn-icon">✏️</button> <button class="c-btn-icon danger">🗑</button>

🗺️ Layout de Telas

Toda tela nova segue este fluxo vertical:

1. Header (sticky)
2. Tabs nav
3. Filtros bar (se necessário)
4. Content area (max-width: 1100px · margin: 0 auto · padding: 28px 24px)
max-width do conteúdo: sempre 1100px com margin: 0 auto. Isso garante legibilidade em telas largas.
